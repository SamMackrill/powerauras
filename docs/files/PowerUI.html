Parameters

  • class: The name of the class to extend.
  • data: The data table of the class to merge in.
  • ...:

Return value:

Returns the constructed class.
PowaAuras.UI:AnonymousWidget (class, data, ...)
Initialises a frame with an anonymous widget class, which extends a class on the fly to create a class tailored to specific needs at runtime. The class is then immediately constructed afterwards.

Parameters

  • class: The name of the class to extend.
  • data: The data table of the class to merge in.
  • ...: Any arguments to pass to the constructor and init functions.

Return value:

Returns the constructed frame.
PowaAuras.UI:BuildFrameFromDefinition (def, parent)
Builds a frame from a table based definition. This allows you to bypass using XML for frame creation while also handling all of the boring SetPoint/SetSize calls that are otherwise present.

Parameters

  • def: The definition to load.
  • parent: The parent frame of the frame.

Return value:

Returns the constructed frame.
PowaAuras.UI:Construct (ui, widget, ...)
Generic constructor function for all UI widgets. Performs the mundane functions of copying the Hooks and values from the class definitions, registering scripts and initialising the widget.

Parameters

  • ui: The PowaAuras.UI table.
  • widget: The frame to be constructed with a class.
  • ...: Arguments to be passed to the Init function of the class, if it exists.
PowaAuras.UI:Register (name, data, isAnonymous)
Turns a widget definition table into metatables with constructor-like functionality.

Parameters

  • name: The name of the widget class to define.
  • data: The table of data to use when defining the class.
  • isAnonymous: Defaults to false, if set to true then the name of the class is not checked and it will not be registered with the UI table.
